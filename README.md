# 📝 윤선생 인스타그램 캡션 어시스턴트

GPT-4를 활용한 윤선생 방문학습 전용 인스타그램 캡션 자동 생성 도구

## ✨ 주요 기능

- **AI 기반 캡션 생성**: GPT-4가 두 가지 스타일의 전문적인 캡션 생성
- **단계별 안내**: 복잡한 캡션 작성을 4단계로 간소화
- **주제별 맞춤 질문**: 선택한 주제에 따라 관련 질문만 표시
- **원클릭 복사**: 생성된 캡션을 클릭 한 번으로 복사
- **반응형 디자인**: 모든 디바이스에서 최적화된 사용 경험
- **보안 강화**: API 키 완전 보호

## 🚀 배포 방법

1. GitHub 저장소 생성 및 코드 업로드
2. Vercel 계정 생성 및 프로젝트 import
3. 환경변수에 OpenAI API 키 설정
4. 자동 배포 완료

## 🔧 환경 설정

### 필요한 환경변수
- `OPENAI_API_KEY`: OpenAI API 키
